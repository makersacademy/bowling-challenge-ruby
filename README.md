### User Stories

```
As a Player,
So that I can keep track of my scores,
I want to be able to record my score for each roll

As a Player,
So that my score is accurate,
I want to receive a bonus for every spare

As a Player,
So that my score is accurate,
I want to receive a bonus for every strike

As a Player,
So that my score is accurate,
I want the game to end after 10 frames

As a Player,
So that I can know how well I played,
I want to receive an overall score at the end of the game
```
