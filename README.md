Bowling Scoreboard app written in Ruby.

To install:
```
$ cd bowling-challenge-ruby
$ bundler
$ rspec --init
```

To use:
```
$ ruby lib/bowling_scoreboard.rb
```
Example input:
```
Frame 1
Enter roll 1 pinfall:
Input = "5"
Enter roll 2 pinfall:
Input = "4"
Frame 1
5 4
Score: 9
```
Example scorecard:

Spares are recorded as /, eg. 5, 5 => " 5 /"
Strikes are recorded as X, eg. 10 => "X"
